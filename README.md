  ## Project Info
  - Part 1 of the project is about capturing user input and validating that input to see if it is        correct
  - There are 3 classes in total of part 1 of the project which are :
  ---
   ## Login Class
  - In this class a users propmted information is validated by checking if :
    1. The users username has an underscore and must have 5 characters in total
    2. The users password has :
    -  An uppercase letter
    -  A digit
    -  It is 8 characters long or more
    3. The users phonenumber has a south african international code

    ## Mainapp class
    - In this class a user is prompted to enter their information which is :
      1. Their Username
      2. Their Password
      3. Their Phonenumber
      - Provided that they follow the rules above in the login class
        their information shall be accepted then captured in the respective variables

    ## LoginTest class
    - This class is used to read each line of code in the respective classes to check if they are          fuctioning as they should
